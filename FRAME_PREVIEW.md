# Mobile Frame Preview

The app is wrapped in a stylized mobile device frame (with notch and side buttons) for presentation.

- Wrapper: `.device-frame` surrounds the `#app` container.
- Notch: `.device-notch` centered at the top.
- Side buttons: `.device-buttons` on the right.

To view:

```sh
python3 -m http.server 5500 --directory "/Users/simonwilke/BPA Mobile Project/non-react-app"
# open http://localhost:5500/index.html
```
